MyScript Shape

  • Technology
  • Shape

    • MyScript Shape is designed for building applications that enable the recognition of shapes created with digital ink. MyScript Shape is ideal for creating custom sketching, diagrams based and note-taking applications through real-time or asynchronous stroke processing.

    Features

    Basic geometric shapes

    MyScript Shape engine enables the recognition of more than 20 basics geometric shapes: triangles, rectangles, squares, circles, ellipses, arrows, arcs, etc.

    Beautification

    MyScript Shape can be used to beautify the presentation and layout of shapes: orientation, horizontal/vertical alignment, connections between shapes for perfectly formed organization charts, diagrams, tables, etc.

    Interactivity

    MyScript Shape enables two use cases: an interactive use case, where the strokes are processed incrementally, for example a drawing application on interactive whiteboards; a non-interactive use case, where a input of shapes and drawings is processed after all its elements have been written.

    Editing gestures

    MyScript Shape supports scratch-out gestures.

    Undo/redo

    MyScript Shape supports undo/redo actions to remove or add the last drawn strokes.

    Supported input shapes

    MyScript Shape can recognize a fixed set of hand drawn shapes.

    A shape is related to one or more primitives. A primitive is the basic geometric unit of a shape: It is either a line segment, an arc, or a decorated line or arc (in the case of arrow heads). Each type of recognized shape can be described by one or more of these basic primitives (for example, a rectangle is made up of 4 line segments, a circle is a type of arc, and so on).
